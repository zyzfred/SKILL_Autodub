# Release Notes: v1.1.0

## English

`v1.1.0` is the current repository release tag for the synchronized multilingual subtitle workflow baseline.

### What This Repository Release Includes

- syncs `video-target-subtitles` to `v1.1.0`
- adds the new `video-subtitle-delivery` skill at `v0.1.0`
- upgrades the repository from a single-skill layout to a two-skill workflow
- adds bilingual repository, skill, and reference documentation
- documents the downstream handoff boundary for multilingual dubbing workflows

### Repository Workflow Baseline

1. `video-target-subtitles` generates multilingual subtitle assets
2. human QA approves subtitle text and timing
3. `video-subtitle-delivery` packages reviewed subtitles for final delivery
4. a separate dubbing workflow may consume the approved assets downstream

### Validation Snapshot

- `video-target-subtitles/scripts/*.py` compile successfully
- `video-subtitle-delivery/scripts/*.py` compile successfully
- repository docs now provide English and Simplified Chinese coverage for all active markdown docs
